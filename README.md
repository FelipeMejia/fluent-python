# fluent-python

Code and exercises while working through *Fluent Python*.

## Contents

- `card_deck.py` — `FrenchDeck` example using `__len__` / `__getitem__` and `namedtuple`.
- `classes-review/` — notes and small scripts on classes, MRO, and cleanup patterns.

## Running

Requires Python 3. For example:

```bash
python3 card_deck.py
```
